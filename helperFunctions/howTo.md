 # How to use the nfl helper functions 

Provided in this folder are functions that will allow you to retreive data by passing the correct arguments.

**helperFunctionsExamples.py** contains the actual helper function code.
you should not need to change this file.

**helperLogic.py** contains examples of how to pass the proper arugments and run the functions along with a brief description on how to use it.
